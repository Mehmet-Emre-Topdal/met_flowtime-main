0. Tasarımın özeti (referans)

Faz 5 — Collections (yeni özellik, tam)
En büyük yeni iş. Mevcut GPS feature'ı şablon alınır.

Tip: src/types/collection.ts → CollectionDto { id, userId, name, color, order, createdAt }. TaskDto'ya collectionId: string | null eklenir.
Firestore: collections/{collectionId} koleksiyonu; Security Rules'a uid kısıtı (CLAUDE.md hybrid mimari kuralı).
API katmanı (RTK Query zorunlu — CLAUDE.md): src/features/collections/api/collectionsApi.ts (list/create/update/delete, Client SDK). Görev oluşturma/güncellemeye collectionId eklenir; CreateTaskDialog.tsx seçili koleksiyonu atar.
Slice: selectedCollectionId (timer/task slice paterni).
UI: sidebar listesi (renk noktası + ad + adet), dashboard başlığındaki dropdown seçici, "+ Koleksiyon" modalı (basit ad girişi, palette'den renk). KanbanBoard selectedCollectionId'ye göre filtrelenir.
Migrasyon: collectionId === null görevler için "Genel/Varsayılan" sanal koleksiyon → mevcut görevler kaybolmaz.
